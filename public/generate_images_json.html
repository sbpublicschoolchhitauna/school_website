images.json बनाने के लिए सरल उपकरण

निर्देश:
  1. नीचे "Choose images" से अपनी सभी slider images चुनें (आप slider_images फ़ोल्डर की फाइलें चुनें)।
  2. फाइल चुनने के बाद "Generate images.json" पर क्लिक करें — एक JSON फ़ाइल डाउनलोड होगी।
  3. डाउनलोड की हुई images.json को अपने website فول्डर में रख दें (index.html के साथ)।
  4. अब आप index.html को ब्राउज़र में खोल कर slider देख सकते हैं (कोई server नहीं चाहिए)।

Tip: आप उसी नाम वाली images को slider_images/ में रखकर index.html के साथ खोलें।
